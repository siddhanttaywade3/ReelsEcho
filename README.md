#  ReelsEcho – AI Reel Generator

ReelsEcho is an AI-powered web application that converts text into engaging voiceovers for short-form video content. It enables users to generate natural-sounding audio from text, making content creation for Instagram Reels, YouTube Shorts, and other social media platforms faster and more efficient.

---

##  Project Overview

ReelsEcho is designed to simplify reel creation by transforming text into high-quality AI-generated speech. Users enter a script, select a voice, and instantly generate realistic voiceovers suitable for social media content.

---

##  Features

-  AI-powered Text-to-Speech
-  High-quality voice generation using ElevenLabs
-  Web-based user interface
-  Fast audio generation
-  User-friendly design
-  Custom text input for voice generation
