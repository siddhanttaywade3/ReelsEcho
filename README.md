ReelsEcho AI is a Flask-based web application that converts text into natural-sounding AI voiceovers using the ElevenLabs Text-to-Speech API. It provides a simple and responsive interface for generating high-quality audio, making it suitable for content creators producing voiceovers for social media videos and other digital content.

Features:
- AI-powered text-to-speech conversion
- ElevenLabs API integration
- Responsive web interface built with HTML and CSS
- Fast and efficient voice generation
- Backend developed using Flask and Python
